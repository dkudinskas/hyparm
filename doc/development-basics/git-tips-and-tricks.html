Git tips and tricks

The repository of the hypervisor is managed through Git. For basic information on how to set up Git, take a look at GitHub Help. If you want a GUI to browse the repository and its history, try gitg or qgit.

Fixing mistakes

To revert uncommitted changes in a single file: 

git checkout -- filename

Merging

To get some branch from a forked repository: 

git remote add fork git://github.com/username/hyparm.git
git fetch fork
git checkout -b somebranch fork/master

To merge the current branch with some branch from a forked repository: 

git remote add fork git://github.com/username/hyparm.git
git pull fork somebranch

Searching

To list all tags that contain a specific commit: 

git tag --contains commit-hash
