ARMv7-A architecture basics

The ARMv7-A architecture is a 32-bit RISC architecture. It features multiple instruction sets:

  • ARM: a fixed-width 32-bit instruction set;
  • Thumb-2: a mixed 16-bit and 32-bit instruction set for increased code density;
  • ThumbEE: a variant of the above for dynamically generated code.
