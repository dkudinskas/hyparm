Overview

Before this branch is usable, some extra steps are required to bootstrap: the VMCG submodule and its submodules must be checked out, and VMCG must be built prior to building the hypervisor. Outline of the required steps: 

# git submodule init
# git submodule update
# cd ext/vmcg
# git submodule init
# git submodule update
# ./setup-gcc
# make -C build

VMCG can be built with GCC ≥ 4.7 or Clang ≥ 3.2 (using the setup-clang script).
