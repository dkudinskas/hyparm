Compile-time configuration

Compile-time configuration is managed through Kconfig. All configuration options are defined in scripts/kconfig.data.

Modifications to Kconfig

The version of Kconfig in the repository is slightly modified to fit the needs of this project. The most important change is that in the generated C header, all symbols starting with CONFIG_DEBUG_ are always defined. The corresponding macros have the value 0 when disabled and 1 when enabled.

Updating Kconfig

The scripts directory contains a script to update the copy of Kconfig in the repository with Kconfig from the Linux kernel.
